MacrosFinder.  
Defines all macros unique definitions with string type body.  
Run:  
```sh  
make  
./main < file  
```  
Run tests:  
```sh  
make test  
```  
For run SH script(do the same thing as the C++ program):  
```sh
./main.sh
```  
For test program run:  
```sh  
./main -t < file  
```  
For compare the result of SH script and C++ program run:  
```sh
make compare
```
